# 🤖 AI Coding Assistant

> Автономный AI-помощник для программирования прямо в терминале.  
> Бесплатно. Без подписок. Работает с любым проектом.

![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)
![Free](https://img.shields.io/badge/API-Free-brightgreen?style=flat-square)

---

## Что умеет

- **Пишет код** на 100+ языках программирования
- **Находит и чинит баги** с объяснением причины
- **Запускает команды** прямо из чата (`run python main.py`)
- **Коммитит в git** с осмысленными сообщениями (`commit исправил баг`)
- **Сканирует проект** и понимает его структуру
- **Помнит контекст** разговора
- **Автоматически** переключается между моделями если одна недоступна

---

## Быстрый старт

```bash
git clone https://github.com/YOUR_USERNAME/ai-coding-assistant
cd ai-coding-assistant
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python assistant.py
```

При первом запуске введи API ключ. Получи бесплатно на [openrouter.ai](https://openrouter.ai)

---

## Команды

| Команда | Описание |
|---|---|
| `run <команда>` | Выполнить shell команду |
| `commit <сообщение>` | Закоммитить изменения в git |
| `scan` | Показать структуру проекта |
| `clear` | Очистить историю чата |
| `exit` | Выйти |
